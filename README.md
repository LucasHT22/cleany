## Run

1. Install Go (1.25+)

Go to [Go Website](https://go.dev/dl)

2. Clone the repo

```
git clone github.com/LucasHT22/cleany.git
```

3. Run
It'll scan C:\ by default, but you can specify the folder:

```
go run . C:\Users\lucas
```
